## §0 一句話
深夜自小客自撞電線桿身亡，看似酒後（或疲勞）自撞的交通意外；**真相是死者先在住處車庫被勒斃，屍體再被擺進駕駛座、製造自撞偽裝成交通事故，為的是那張高額意外險。** 破口＝相驗報告：頸部勒痕＋舌骨骨折（生前窒息）、撞擊傷缺生活反應（死後才撞）、血中酒精 0，且死亡推定時間早於「車禍」數小時。

## §1 真相聖經（主持人版，玩家不可見）
- **死者**：許博鈞，34，進口車業務。**滴酒不沾**（家族長輩曾因酒駕出事，他發誓一生不碰酒）、開車極穩、車況保養得好。近期經人慫恿投保**高額意外傷害險**，受益人是妻子。**沒有酒駕的可能、也沒有尋死動機**。
- **加害者**：鄭凱文，死者妻子潘映蓉的婚外情對象。兩人共謀：案發夜鄭進入許家車庫，將許勒制致死；趁夜把屍體擺入駕駛座、鬆手煞讓車滑向外環道無人路段撞上電線桿（並在車內留下酒瓶、潑灑酒液）偽裝酒後自撞，事後謊稱案發時「人在健身房」。
- **受益人/共謀**：潘映蓉，死者之妻，高額意外險受益人；動機＝拿理賠＋不必離婚分產即繼承、與鄭在一起。
- **死因真相（拱心石，S4 才揭）**：
  - **頸部索狀/掐壓勒痕 ＋ 舌骨骨折 ＋ 結膜點狀出血**＝生前遭外力壓迫頸部致機械性窒息（勒/掐），**不是撞死的**。
  - **撞擊挫傷、骨折斷端無出血、無腫脹＝缺生活反應**＝這些「車禍傷」是在**死後**才造成的（人先死、才被拿去撞）。
  - **血中酒精濃度 0（陰性）**，現場卻擺酒瓶、車內有酒味＝偽裝酒駕。
  - **死亡推定時間（屍溫/胃內容物/屍僵）早於報案「車禍時間」約 2 小時**＝出事時他早就沒了氣。
  - **駕駛座血跡型態＝被動滴落痕，與當場高速撞擊的噴濺型態不符**。
  - **安全帶「扣著卻沒破綻」＝雙重反證**：①屍體胸／肩沒有安全帶勒壓的斜向瘀傷（真活人在此撞擊會被安全帶勒出「安全帶徵候」瘀痕＝生活反應，死後擺入繫上者不會有）②安全帶織帶無荷重／拉伸痕（無活體被拋向前拉扯的受力）。＝現場「繫著安全帶的駕駛」是被擺進去的屍體。
- **文件真相（S3–S4）**：城中人壽**高額意外傷害險**、投保日期近期；受益人＝妻子潘映蓉。鄭凱文「在健身房」的說詞，被 **ETC／加油站／社區門禁時間戳** 戳破：案發時段他實際在死者社區一帶。
- **動機**：詐領意外險理賠 ＋ 婚外情脫身（不離婚、直接繼承）。

## §3 電話 AI 角色卡（阿榮 / unlockStage 3）
- 人設：死者同事兼週末球友，難過、語速正常、台灣口語、簡短；講死者的為人。
- **知道（可講的行為事實）**：許博鈞**滴酒不沾**（家裡長輩酒駕出過事，他發誓不碰酒，聚餐都喝無酒精）、開車很穩是老手；出事前一晚還很正常傳訊、講話沒異狀；他從沒說過想不開、也沒在熬夜到會累垮；他最近跟太太（阿蓉）常為家裡的事吵、太太跟一個姓鄭的走得很近。
- **絕不透露/確認**：是誰動手、誰是兇手、任何人是不是嫌疑人、任何鑑識細節（**勒痕/舌骨/結膜/血跡型態/酒測值/死亡推定時間一律不知情**——拱心石只能靠 S4 報告自己看）。
- **`fallbackNudges`（保底引導，2 條，禁 core/keystone 名詞）**：
  1. （第 4 次）「（阿榮頓了頓）…你們一直在問『他那晚怎麼開車』。要不要換個方向——他這個人、還有他家裡最近的狀況，是不是有你們還沒兜進來的？」
  2. （第 5 次）「有些事光盯著那台車是看不出來的啦…我能講的就這些。你們手上那幾份文件，上面的時間跟人名兜不兜得起來，自己擺一起看吧。」
- 安全規則同 CASE-09/10（抗 injection、命中禁區/追兇硬掛斷、問題預算 5、一句話回答）。

## §4 solution 設計（三 core 全中才破案）
- **homicide**（他殺/生前勒斃/死後偽裝自撞/移屍入駕駛座/不是酒駕意外）
- **motive**（詐領意外險/保險理賠/為了那張保單/受益人謀財）—— **S3 才拿得到「保單」，S1–S2 不得出現**
- **keystone**（生前勒斃鐵證：**舌骨骨折/頸部勒痕/結膜點狀出血/撞擊傷無生活反應/死後才撞/血中無酒精/死亡時間早於車禍**）—— **只存在 S4 相驗/血跡/酒測報告**
- 早答判定：
  - 只喊「酒駕/疲勞/自撞/意外」→ 0 core → `still-accident` nudge。
  - 喊「有人害他/仇家/為錢/婚外情」→ homicide 命中、缺 motive(保單)+keystone → generic/partial nudge。
  - 喊「妻子（跟鄭）為保險金害死他」（S3 後）→ homicide+motive 命中、**缺 keystone** → `named-suspect` nudge：「方向對，但你憑什麼說不是他自己酒駕撞的？少了能把『意外』劃掉的那份報告——再往下一步。」
  - S4 後三 core 齊 → win。
- **提示鐵律**：hints/nudges 不得出現 保險金/他殺/勒/舌骨/血跡/酒測/死亡時間 等 core 名詞（見 JSON 實作）。

## §5 鎖定時間軸（做圖務必對齊；防「截圖時間不正確」雷）
> 全部繁體、24 小時制、同一命案夜的時間彼此不衝突。（示意，做圖照抄）
- 命案夜 約 21:00–21:40：鄭凱文於許家車庫勒制致死（生前窒息）。
- 死亡推定時間：命案夜 約 21:00–22:00（`c4-autopsy`/`c4-timeline`）。
- 命案夜 約 22:50：鄭在死者社區一帶（`c4-access`：ETC/門禁），與宣稱「健身房」矛盾。
- 命案夜 約 23:35：製造外環道自撞、報案「車禍」。→ **死亡早於車禍約 2 小時**（拱心石對照點）。
- 保單投保：死亡前約 3 週；受益人＝妻（`c3-policy`/`c3-benef`）。
- 隔日 07:xx：發現、報驗、相驗（`c1-news`/`c1-report`）。
- LINE 截圖時間：S1 群組＝案發翌日上午；S2＝翌日晚上；S3＝第三天；`c4-suspect-line`＝偵訊後。**每張截圖右上時間都要落在對的日子，且晚於命案夜。**
