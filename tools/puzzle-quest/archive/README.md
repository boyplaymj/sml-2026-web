## 加封存的動作（SOP）
1. 該題 `title` 前面加 `【封存】`（DDB item + 本地 json 都改）。
2. 在上表補一列：id／標題／階段／**玩過結論**（哪裡好、哪裡待改）。
3. 把這題暴露的**新踩雷**寫進 `LESSONS.md`。
4. 該題引擎檔留原地，不搬。
