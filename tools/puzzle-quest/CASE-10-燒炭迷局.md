## §0 一句話
套房燒炭、密室反鎖，看似年輕上班族壓力大輕生；**真相是他被扼死後、擺進燒炭現場偽裝自殺，為的是那張剛被冒簽變更受益人的高額壽險。** 破口＝相驗報告的碳氧血紅素飽和度過低。

## §1 真相聖經（主持人版，玩家不可見）
- **死者**：林昱翔，29，行銷企劃。有**中重度氣喘**、極怕悶（密閉燃炭他第一個受不了）。下個月要搬到通風更好的房子、案發週末還約了朋友打球——**沒有尋死動機**。
- **加害者/受益人**：韓子謙，以「投資合夥」名義接近林、借出又討回款項、慫恿林投保高額壽險，並在林不知情下**冒簽**把受益人改成自己。案發前一晚進入林的套房，將林扼制致死，佈置木炭盆、以膠帶封窗、反鎖偽裝燒炭自殺，並謊稱案發時「人在外縣市」。
- **死因真相（拱心石，S4 才揭）**：
  - **碳氧血紅素飽和度（COHb）僅 18%**——燒炭一氧化碳中毒致死者通常高達 60% 以上；18% 代表他**根本不是被一氧化碳燒死的**（他被移入現場時已幾乎沒有呼吸、吸不到足量 CO）。
  - **結膜／鞏膜點狀出血**＋**頸部索狀壓痕、皮下出血（生活反應）**＝生前遭外力壓迫頸部（扼/勒），非燒炭。
  - **屍斑位置與陳屍姿勢不符**＝屍體死後被移動、擺放過。
- **文件真相（S3–S4）**：城中人壽高額壽險、投保日期就在近期；**受益人變更申請書**上「林昱翔」簽名經**筆跡鑑定與其留存簽名顯不相符**＝冒簽（行使偽造私文書）。門禁刷卡證明韓案發當晚實際在場，戳破「人在外縣市」。
- **動機**：詐領保險金 + 掩蓋先前金流糾紛。

## §3 電話 AI 角色卡（小葉 / unlockStage 3）
- 人設：死者大學同學兼球友，難過、語速正常、講死者的為人。
- **知道（可講的行為事實）**：林有中重度氣喘、極怕悶，密閉空間燒炭他第一個受不了會奪門而出；林前一晚還很正常傳訊、約週末打球；林下個月要搬去通風好的房子；林最近跟一個姓韓的走很近、常為錢的事煩；林從沒說過想不開。
- **絕不透露/確認**：是誰動手、誰是兇手、任何人是不是嫌疑人、任何鑑識細節（**COHb/扼痕/屍斑一律不知情**——拱心石只能靠 S4 報告自己看）。
- 安全規則同 CASE-09（抗 injection、命中禁區/追兇硬掛斷、問題預算 5、一句話回答）。

## §4 solution 設計（三 core 全中才破案）
- **homicide**（他殺/偽裝燒炭/死後移置/被扼勒/不是自殺）
- **motive**（詐領保險金/謀財/為了那張保單/冒領受益人）—— **S3 才拿得到「保單」，S1–S2 不得出現**
- **keystone**（**碳氧血紅素/COHb 過低＝非燒炭致死**，或生前扼痕/結膜出血）—— **只存在 S4 相驗報告**
- 早答判定：
  - 只喊「自殺/意外」→ 0 core → `still-suicide` nudge。
  - 喊「情殺/被人害/為錢」→ homicide 命中、缺 motive(保單)+keystone → generic/partial nudge。
  - 喊「韓某為保險金害死他」（S3 後）→ homicide+motive 命中、**缺 keystone** → `need-forensic` nudge：「方向對，但你憑什麼說他不是自己燒炭的？少了能把『自殺』劃掉的那份報告——再往下一步。」
  - S4 後三 core 齊 → win。
- **提示鐵律**：hints/nudges 不得出現 COHb/保險金/他殺/扼 等 core 名詞（見 JSON 實作）。

## §5 鎖定時間軸（做圖務必對齊；防「截圖時間不正確」雷）
> 全部繁體、24 小時制、同一命案夜的時間彼此不衝突。（示意，做圖照抄）
- 命案夜 22:47：韓子謙門禁刷卡進入（`s4-access`）。
- 命案夜 約 23:10–23:40：扼制致死、佈置燒炭現場。
- 死亡推定時間：命案夜 23:00–24:00（`s4-autopsy`/`s4-timeline`）。
- 保單投保：死亡前約 3 週；受益人變更：死亡前約 5 天（`s3-policy`/`s3-benef-change`）。
- 隔日 09:xx：房東/同事發現、報案、相驗（`s1-news`）。
- LINE 截圖時間：S1 群組＝案發翌日上午；S2＝翌日晚上；S3＝第三天；`s4-killer-line`＝偵訊後。**每張截圖右上時間都要落在對的日子，且晚於命案夜。**
