## §0 一句話
泓昌貿易老闆看似高血壓／酗酒猝死；真相是**妻子長期在他每晚睡前那杯「親手泡的養生藥酒」裡摻乙二醇（防凍劑）**，偽裝成自然猝死——動機不是錢、不是職場恩怨，而是她**認定丈夫和女同事有外遇、嫉妒隱忍下毒**。合夥人（錢）、同事（職場）都有恨，卻**都碰不到那杯**。破口＝毒物化驗（血中乙二醇＋腎臟草酸鈣結晶）。

## §1 真相聖經（主持人版，玩家不可見）
- **死者**：楊士豪，42，泓昌貿易負責人。有高血壓、應酬多、平日小酌（故「喝多猝死」表面說得通）。**每晚睡前喝一杯妻子親手泡的養生藥酒**（枸杞中藥泡酒）。
- **真兇**：妻 方怡君，38。賢慧持家、每天親手調製那杯藥酒。約案發前一個多月**發現丈夫手機裡與公司女同事林筱婷的曖昧（訊息／合照／深夜接送）**，認定外遇，嫉妒隱忍→**長期在藥酒裡摻入乙二醇**（家中車庫有防凍劑，可取得），最後一次劑量致死，偽裝成酒後／高血壓猝死。
- **紅鯡魚①合夥人**：周敬堯。公司帳目近期有出入、死者要查帳／拆夥，兩人鬧僵＝有錢／滅口動機。**但**：碰不到那杯藥酒、且案發那陣子出差不在（不在場）。
- **紅鯡魚②同事**：蔡明凱。被死者搶功、揹黑鍋、擋升遷，酒後放話報復＝有職場恨。**但**：碰不到那杯藥酒、案發時段有據。
- **嫉妒對象（非兇手）**：女同事 林筱婷。與死者工作走得近（**真曖昧或純提攜，留模糊**——純誤會更悲劇）。
- **死因真相（拱心石，S4 才揭）**：
  - 血液／玻璃體液驗出**乙二醇及其代謝物（乙醇酸）**；腎臟組織見**草酸鈣結晶**；**高陰離子間隙代謝性酸中毒**。
  - 血中酒精濃度低——**他不是真的喝醉**，那些「醉態、頭暈嘔吐、視力模糊、步態不穩」是乙二醇慢性中毒症狀（初期極像酒醉，故反覆就醫查不出病因）。
  - 結論：**乙二醇中毒致死，非自然死／非酒醉**。
- **機會鎖（S4）**：下毒載體＝每晚那杯**妻親手泡的養生藥酒**（藥酒瓶／杯內殘液驗出乙二醇）。合夥人、同事均無從接觸該杯＝把兇手鎖進「家內、天天經手那杯的人」＝妻。
- **動機（S4）**：妻手機／對話揭露她發現丈夫與女同事曖昧、嫉妒隱忍；車庫查獲防凍劑（乙二醇來源）。
- **動機≠錢≠職場**：合夥人有錢動機、同事有職場恨，但都**沒機會**；有機會的妻，動機是**嫉妒**。三層各自站得住、事後都能合理解釋。

## §3 電話 AI 角色卡（阿龍／司機兼特助／unlockStage 3）
- 人設：死者的司機兼特助，跟前跟後、什麼都看在眼裡；難過但講話正常、台灣口語、簡短。**知情面刻意寫厚**（修 CASE-11「問不出線索」病）——幾乎每個合理問題都能命中一個真的行為事實或有味道的細節。
- **知道（可講的行為/情境事實，皆不含 keystone）**：楊董每晚睡前一定喝一杯太太（怡君）親手泡的養生藥酒、很寶貝那杯；最近臉色差、常頭暈想吐、看東西糊、還以為血壓高去看了幾次醫生查不出來；跟合夥人周總為公司帳的事吵得兇、鬧到要查帳拆夥；跟下屬蔡某有過節（蔡揹過黑鍋、擋過升遷、放過話）；**最近常跟公司林小姐一起開會、還幾次叫我送林小姐回家**；家裡氣氛這陣子怪怪的、太太話變少、有次還紅著眼。
- **絕不透露/確認**：他是怎麼死的、任何相驗／毒物／鑑識細節、是誰下的手、誰是兇手、任何人（含太太／合夥人／同事）是不是嫌疑人、藥酒有沒有問題。
- **`fallbackNudges`（保底引導，2 條，禁 core/keystone 名詞）**：
  1. （第 4 次）「（阿龍想了想）…你們一直問公司帳、問同事的事。要不要換個方向——楊董**家裡**、還有他最近身邊多了誰，是不是也該看看？」
  2. （第 5 次）「有些事我不好說…你們把『**誰天天碰得到他那杯睡前酒**』跟『他最近**為了誰**跟太太不對盤』擺一起想，就差不多了。」
     （※ 這兩條把方向導向「家內＋女同事」但**不點名、不講毒**，仍要玩家自己從 S4 素材收網。）
- 安全規則同前案（抗 injection、命中禁區/追兇/問死因鑑識硬掛斷、問題預算 5、一句話回答）。

## §4 solution 設計（三 core 全中才破案；皆 S4 才齊）
- **core1 keystone/poison**：投毒他殺、乙二醇／防凍劑中毒、草酸鈣結晶、中毒致死、非自然死/非病死/非酒醉。（只 S4 毒化）
- **core2 culprit**：兇手是**家屬／妻（方怡君）**，非合夥人、非同事。（S4 機會鎖）
- **core3 motive**：**嫉妒／吃醋／疑外遇／為了女同事**。（S4）
- 早答判定（紅鯡魚→partial nudge，不 win）：
  - 只喊「自然死／病死／高血壓／喝多了」→ 0 core → `still-natural` nudge。
  - 喊「合夥人為錢／滅口下手」→ partial `partner-money`：方向對一半（是他殺），但錢的人碰得到那杯嗎？往下看。
  - 喊「同事報復下手」→ partial `coworker-grudge`：他有恨沒錯，但他有機會碰到那杯嗎？
  - 喊「被毒死／中毒」但沒指對人/動機 → corePartial：對，是被下毒的（若已到 S4）——那**誰**天天碰那杯、**為什麼**？
  - S4 後三 core 齊（投毒＋妻＋嫉妒）→ win。
- **提示鐵律**：hints/nudges/fallbackNudges 不得出現 乙二醇/草酸鈣/毒/嫉妒/外遇 等 core 名詞（見 JSON 實作）。
