## §0 一句話

- **死因真相（拱心石①·method，S4 才揭）✅已定**：現場布置成「加班獨處＋老舊金屬檢視燈＋地面積水」的觸電意外，實為高博彥動手腳讓那盞金屬燈**外殼整個帶電**、又**同時破壞掉保護接地**，沈一碰、透過腳下積水導到地就致命。兩個鐵證證明是「人做的」不是「用舊的」：

## §9 🎙️ 語音線索台詞 + 情境（使用者變聲器錄；≥3，一階段一段）
> 規則：全繁體口語、**不得出現 core/keystone 名詞**（贗品/假畫/洗錢/接地/電線/兇手/滅口/鈦白…一律不准）。使用者用變聲器錄不同角色 → 我用靜態 ffmpeg 微調（電話帶通/信箱嘟聲/殘響/pitch，見 `reference_audio_clue_pipeline.md`）。三角色各給不同 pitch 區隔。

### 語音①（S1｜`m1-voicemail.ogg`｜死者 沈維鈞 → 老友/信任的學生 · 語音信箱）
- **情境**：出事前 2–3 天深夜，沈獨自加班後打給多年老友（對方沒接、留言）。剛發現那幅畫「不對」，燙手、還不敢張揚。
- **語氣**：疲憊、壓低、凝重，欲言又止；像挖到不該挖的東西。
- **台詞**：
  > 「喂……是我，沈維鈞。這麼晚打給你，不好意思。那幅畫……我越看越不對勁，有幾個地方，怎麼兜都兜不起來。我得把報告好好寫清楚——寫完，再當面跟你講。這事……恐怕不小。你先別跟別人提，改天我約你。」
- **變聲/後製建議（我處理）**：男聲、偏低沉疲憊；`rubberband pitch≈0.85`＋電話帶通（highpass 300 / lowpass 3200）＋信箱前置嘟聲。

### 語音②（S2｜`m2-voice-buyer.ogg`｜藏家 郭崇德 → 沈維鈞 · 怒氣語音信箱）
- **情境**：案發前幾天，大藏家郭崇德得知沈可能對他手上/關注的藏品出不利鑑定、將重挫行情，暴怒留言施壓。（金錢紅鯡魚）
- **語氣**：財大氣粗、不耐、帶威脅；音量大、語速快。
- **台詞**：

## §10 📓 手寫筆記本三頁（使用者實拍手寫；卓文瀚筆記本＝紅鯡魚載體）
> 走**使用者親手寫＋實拍**（同一本、同一支筆、同一手跡＝三頁自然一致，最真；比照 CASE-11 安全帶真圖）。我收圖後套輕度「翻拍／採證」處理（泛黃、頁緣陰影、CASE FILE 邊框選用）。全繁體、**不得出現 core/keystone**（贗品/洗錢/電/兇/報告內容都不准），只能是「有怨氣的助理＋想找沈談＋記了沈的時間」這種曖昧料。
> **拍攝**：每頁單獨平拍、光線平均、字寫滿一點（密密麻麻感）；可留塗改、劃線、字跡潦草＝更真。三頁對應 `m1-notebook-a`(S1)／`m2-notebook-b`(S2·要命頁)／`m3-notebook-c`(S3·翻案頁)。

### 頁①（`m1-notebook-a`｜S1·微種子：有怨氣的助理）
> 語氣：碎念、不平，但還不像要出事。可日期標「9/8」之類。
```

又來了。這次「谿山」的複驗實際上是我做的，

升等名單又沒有我。第三年了。
沈老師今天也是弄到快十一點才走，天天都這樣。
```

### 頁②（`m2-notebook-b`｜S2·要命頁：像預謀＋跟蹤，最兇）

```
週二 全天在館、午休出去一趟
週三 晚上大概又是他一個人留到最後
就今晚。不能再忍。
```

### 頁③（`m3-notebook-c`｜S3·翻案頁：其實是崇拜＋求助，無害）
> 語氣：把頁②的「狠」翻軟——原來「講清楚」是想求沈幫忙掛署名、放行升等；「記行程」是想找他落單好好談。可日期標「9/16 深夜」或隔頁。
