# 🎴 甜甜神社 — 御神籤（おみくじ）籤詩池 · 預覽稿 v2

> 御祭神＝**麻雀大明神（牌神）**。**籤詩體例參考「淺草寺觀音一百籤」（源自天竺靈籤）**：**五言四句籤詩 ＋ 吉凶階 ＋ 逐項解**，古典語感、末句常帶轉折或勸誡，牌神/麻將意象融入。
> **先預覽核可 → 再灌入 `sweetbot-shrine-omikuji-pool`。** 每階正式種子擬 **3 張**（本檔先各出 **1 張** 定調）；核可後補滿至 33 張。全 $0（手寫、可後台增修）。

## 規則速記（見 DESIGN §2.3）
- 無限抽、**每抽 100🦷**；**只有當日首抽計運勢**（台北時區日界），第 2 抽起照扣但純看籤詩爽（牙 sink）。
- 首抽抽到**凶類**→ 必須手動【結ぶ】；不綁 → 每日大量扣運氣值（綜合運/body 各 −15，大凶 ×2）。
- 每張籤 6 分項 score → 六軸 buff：`商賣→財運 / 爭事→勝運 / 學問→智運 / 健康→body / 戀愛→人緣 / 旅行→行運`（對應淺草寺的求財/訴訟/功名/疾病/婚姻/行人）。
- 抽到即出**籤紙圖**（每階一張預生成、毛筆字合成）＋ embed 籤詩全文（詩曰四句＋解曰＋分項）。
