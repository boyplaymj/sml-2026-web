# 🎴 甜甜神社 — 御神籤（おみくじ）籤詩池 · 預覽稿

> 御祭神＝**麻雀大明神（牌神）**。籤詩以牌神信仰鋪陳，語氣莊嚴帶麻將梗。**先預覽核可 → 再灌入 `sweetbot-shrine-omikuji-pool`。**
> 每階正式種子擬 **3 張**（本檔先各出 **1 張** 定調性）；核可後我補滿至 33 張。全 $0（手寫、可後台增修）。

## 規則速記（見 DESIGN §2.3）
- 無限抽、預設免費；**只有當日首抽計運勢**，之後純看爽。
- 首抽抽到**凶類**→ 必須手動【結ぶ】；不綁 → 每日大量扣運氣值（綜合運/body 各 −15，大凶 ×2）。
- 每張籤 6 分項 score → 六軸 buff：`商賣→財運 / 爭事→勝運 / 學問→智運 / 健康→body / 戀愛→人緣 / 旅行→行運`。
- 抽到即出**籤紙圖**（每階一張預生成圖）＋ embed 全文。
