# 🧿 甜甜神社 — 御守каталог(真實神社御守清單 → 六軸對應)

> 依真實日本神社實際授與的御守整理,對應到神社運氣**六軸**(💰財運zaiun / ⚔️勝運shengun / 📚智運zhiun / 🩺厄除body / 💗人緣renyuan / 🧳行運xingyun)。
> 每種御守=持有中對某軸 +boost(黑箱、玩家看不到數字)、效期 365 天、花牙齒授與。價分三階(config 可調)。
> 這份是 omamoriTypes 的擴充來源(目前 config 只有 6 種基礎守 → 依此擴到 ~30 種)。

## 🛠️ 對建置的含意
1. **type 很多(~30)→ config.omamoriTypes 大擴充**:每筆 `{ axis, boost, fee }`(顯示名 map 也擴)。
2. **UI:Discord select 上限 25 選項** → 授與所「請御守」要**先選類別(六軸/縁起物)再選御守**(兩層 select),或分頁。**建議兩層**:先選「どの御利益?」(6 軸+縁起物)→ 再列該類御守。
3. **階/價**:普/上/特三階(config 可調);奧社限定另走 §5.2.2。
4. 隨機軸的「開運守」等特殊守:先綁固定軸(如行運),真·隨機留擴充。
