# 模擬麻將館 — 進行方式(玩法流程)討論記錄

> 目的:開工前把「這款怎麼玩」整條走通,減少重工。逐項定案累積於此。
> 對應:內容清單 `CONTENT.md`、機制 `DESIGN.md`。
> 開始:2026-07-11。

## 決策 1:時間模型 ✅
**真實時間 1:1**。遊戲內時間 = 真實時間。
- 時段尖峰 = 真實時段(晚上就是店裡尖峰)。
- 存活天數 = 真實天數。
- 賽季 = 真實幾週,配合 SML 直播賽季。
- 理由:玩家幾乎同一時區(台灣),化解「深夜尖峰不公平」缺點;存活/賽季超直覺。
- 惰性結算按真實經過時間積分(掛機)。

## 決策 2:委派 × 員工能力(自動運轉的引擎)✅
- **設定類政策**(數字,存著自動套用):營業時間、檯費/價格、排班表、進貨目標量。
- **執行類委派**(要有人去做,受能力把關):實際叫貨、補貨、處理事件… 員工能力不足 → **交付失敗**(沒叫到貨→缺貨等)。
- **進貨 = 玩家決定**:親自叫貨(需在線),或設目標量委派店長叫(受店長能力 gate)。
- **連鎖代管**:每間分店靠一位**店長**代管,店長能力決定你不在時這間跑得多好 → 這是「連鎖不限間數」能掛機的引擎(早期微操→養強店長→放手擴張)。

## 決策 3:員工系統(招募/能力/特質/安全)✅

### 3a. 角色與能力細項(公開,面試看得到)
> 麻將館**沒有荷官**(客人自己打)。修正後角色:
- **店長**:管理 / 叫貨議價 / 排班 / 代管分店(委派引擎)
- **店員**(4 細項):開檯(分桌+啟用權限)/ 送餐服務 / 清潔(環境+廁所)/ **湊咖匹配**
- **廚房**:廚藝 / 出餐(有餐飲才需要)
- **保全/顧場**:鎮場處理鬧事

### 3b. 收入修正
- 麻將館合法收入 = **檯費/鐘點(桌子使用時間)+ 餐飲**;客人之間自己玩,店家不抽賭金。
- (原「抽水率」作廢,改「檯費/鐘點費」,強化非賭博定位。)

### 3c. 🀄 湊咖系統(麻將館招牌機制)
- 一桌要 4 咖;單獨/3缺1 的客人靠店員湊桌。
- 店員**湊咖能力 + 工具(LINE 群 / 兩雀 APP)**越強 → 空位填越快 → 檯費越多;湊不到 → 客人等不耐就走。
- **兩雀 APP = 遊戲內匹配工具**,升級它 → 湊咖更快(兩雀數據的最自然接點)。

### 3d. 公開 + 隱藏特質 + 面試判讀(招募核心玩法)
- **公開**:職務能力細項(上面)+ 薪資 + 資歷。
- **隱藏特質**:負面(愛找藉口=機率交付失敗、愛遲到=空班、手腳不乾淨=偷金庫、玻璃心、騎驢找馬易被挖)/ 正面(認真、忠誠抗挖角、人緣拉客、機靈、耐操)。
- **面試 = 自我介紹判讀**:每人附一段自我介紹,藏**線索(tells)**,可學會判讀但**帶雜訊**(有技巧的賭)。
- **真相揭露**:錄用後隨上工**逐步顯現**(日誌/事件驗證你的判斷)。
- **面試供給(沿用便利商店模式)**:每次開招募,系統依**店鋪地點**丟 **3 名面試者**;地點影響應徵池品質(好地段人才好請、偏僻/低價區素質差雷率高=選址另一層權衡)。
- **防洗員工**:開招募要**面試成本**+一次只 3 人+重抽再花錢 → 賭人有代價。(另可加試用期低成本解僱。)

### 3g. 🔴 營業硬條件:沒人=開不了門(但照付租)
- 營業時間內**每個時段必須排到最低人力**,否則該時段**無法營業**(不進客、零收入),**但店租照付**。
- **空班同理**:排了人但員工遲到/沒到(愛遲到特質)→ 該時段一樣開不了 → 損失+付租雙重傷。
- 效果:排班從「效率高低」升級成「**沒排好就流血**」的求生條件(請不到人/排不滿/用到雷 = 付租卻沒生意)。

### 3f. 🧹 環境衛生迴圈(清潔的因果鏈)
- 衛生值 = **會隨營業人流+時間下滑的狀態**(廁所掉更快),不是一次性。
- 店員「清潔」工作補回(需排班有人 + 清潔能力夠)。
- 沒顧好 → 髒亂累積過閾值 → 環境評價↓ → **聲譽↓ → 客流↓**(惡性循環,客訴變多)。
- **高端客層(大戶)對髒亂最敏感、最先跑**;散客較能忍 → 想接大戶就得維持乾淨。
- 尖峰人流大 → 衛生掉更快 → 尖峰要排足清潔,和湊咖/送餐**搶人力**(排班取捨)。

### 3e. 🚨 安全 / 鬧事系統(雙道防線 + 性別不對稱)
- 鬧事事件:警察臨檢 / 打架鬧事 / 醉客酗酒 / 賭債糾紛 / 踢館。
- **預防**(降發生機率):👩女性店員在場、監視器、好環境氛圍。
- **處理**(發生後鎮場):👨男性店員/保全的鎮場能力。
- **性別不對稱**:女性↓鬧事機率但鎮場弱;男性↑鎮場但無預防 → 最佳解=混編。
- 4 項核心技能(開檯/送餐/清潔/湊咖)本身**不分性別**,能力因人而異;性別只作用在「預防 vs 鎮場」這條安全軸。

## 決策 4:定價系統(檯費 → 客量 × 客層 × 風險)✅
- 檯費玩家自訂滑桿,太貴/太便宜都反應在客量與客人**行為**。
- **低價**:客量↑但客層劣化(滋事者↑大戶跑)、衛生掉更快、鬧事機率↑、**警察臨檢/消防檢舉更頻繁**、可能超載。
- **高價**:客量↓(空桌風險)、客層升級(大戶/雀友)、安靜乾淨低風險、湊咖要更主動。
- 定價要**配區域客層**(豪宅區賣超便宜→引雜客趕跑大戶);沒有絕對最佳,找甜蜜點。
- **隱含容客舒適上限**:椅子/空間/廁所(設備)定舒適容客量;硬塞超過→衛生/安全/消防全爆。
- **新增事件**:消防檢舉(超載/安全),與警察臨檢並列,低價塞太多人時觸發率飆高。
- 湧現的館「性格」:薄利多銷(吵雜髒亂高風險) vs 高檔精緻(安靜乾淨)。

## 決策 5:排班 / 營業時間 / 自助開檯 ✅
- **四班制**:早/午/晚/深夜(對齊時段尖峰),每班=一個排班格。
- **排班**:把員工指派到「時段×職務」(開檯/湊咖/送餐/清潔/顧場);每營業時段須達最低人力否則開不了門(決策 3g)。
- **人力搶奪**:員工不能分身,尖峰要覆蓋多職務→人不夠就取捨或多請人(多薪水)。
- **委派店長排班**:可交店長自動排(受其「排班」能力 gate)。
- **薪資只算排到的班**;連排太多/大夜→士氣掉→效率降、雷特質更易發作。
- **營業時間彈性**:開業可選開哪幾班,最多 24h(全 4 班)。開越多=客流/檯費時數越多但人力/薪資/租金時數越重。
- **深夜班**:吃大戶夜貓、檯費高,但要大夜津貼(薪貴)+風險高(酗酒鬧事)+士氣掉更快→24h 是有代價的承諾。
- **🤖 自助式開檯(老闆等級解鎖+裝設備)**:牌桌自助啟用,不再需排人顧「開檯」→釋放人力去別職務或整體更精簡。小取捨:自助抓不到滋事/醉客(安全預防略降)、少服務感→是成長獎勵非全贏。

## 決策 6:開場劇情 + 貸款求生(主線骨幹)✅
- **劇情/舞台**:跟銀行貸款來 **天王里** 開麻將館創業。天王里=遊戲世界,各行政區(夜市邊/商辦/巷弄…)是天王里的區塊。
- **起始資金**:貸款把一筆牙齒存進**館內金庫(營運資金)**→ 買設備/請人的本錢。
- **定期還款**:每期從館內金庫扣本息=**核心求生時鐘**;賺錢先顧還貸。
- **還清貸款=階段勝利**→ 由求生轉為擴張致富/衝排行。
- **還不出=輸(違約破產,本局結束)**:老闆等級歸零、名下館全失;**保留個人牙齒錢包+成就徽章**(對齊賽季傳承保留規則);可重新貸款再創業(roguelike 式創業→破產→再來)。
- **⚠️ 提款防呆(堵鑄幣漏洞)**:貸款未清時,提款只能提「金庫**超過未償本金**的淨利」→ 借來的本金鎖在生意裡,違約只會虧掉自己投入的,無法「借了提走再賴帳」套利。
- **層級關係**:創業 run(貸款週期,可違約後再借)在**賽季**之內;賽季結束才走傳承 reset。
- **數值定案**:①還款週期=**每週(真實 7 天)**,與賽季週對齊 ②**有低利**(提早還清有價值、加壓不絕望)③開分店**可追加貸款(加碼)**→ 每筆疊加還款義務,過度擴張=過度槓桿=風險 ④初始貸款額/每期金額=**暫定值、上線實測 tune**(和租金/收入曲線一起平衡)。

## 決策 7:分店管理(總部面板)✅
- **開分店門檻=現有店證明會賺**(核心):非達等級就能開,而是現有店**還清初始貸款 + 淨利/淨資產達標**才夠格開下一間;**逐店解鎖**(第2間站穩才能開第3間…),連鎖一階階賺出來,不能無腦狂開。
- **開店資金**:可加碼貸款(銀行看第1間獲利才肯貸)or 自有資金。
- **總部面板**:`!模擬麻將館` 先看總部→名下分店一覽(區域/金庫/存活天/狀態/待處理);點進單店管理,`↩返回總部`。
- **店長代管**:每間派店長,能力決定跑多好;出大事(還款危機/鬧事/缺人)才 @ 玩家進去救。
- **槓桿風險**:開越多、加碼貸越多→還款義務越重,一間爆掉可能拖垮全盤。

## 決策 8:財務 / 融資事件(搞錢機會,接貸款經濟)✅
> 事件不只阻撓,也有搞錢機會。與貸款主線咬合:
- 天使投資(注資,可能要分潤/條件)/ 銀行降息(利率下調減壓)/ 政府補助・標會・廠商賒帳(正規小額周轉)。
- 🎰 **協助賭場鋌而走險收資**(灰色):幫地下賭場周轉換快錢,高報酬高風險(臨檢/被抓/黑吃黑/毀聲譽)。
- 地下錢莊(高利貸):急用救命 or 送命,來錢快、利滾利可怕。
- **味道**:玩家開正當麻將館,但蛇蠍誘惑來敲門→為擴張要不要鋌而走險=道德/風險抉擇(遊戲本身仍非賭博,只是被誘惑)。
- 呈現:部分隨機觸發、部分玩家可主動去找(找投資人/上錢莊)。
