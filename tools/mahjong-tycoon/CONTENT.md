## E. 門市人員(職務 × 能力細項 × 隱藏特質)※麻將館無荷官
- **店長**:管理 / 叫貨議價 / 排班 / 代管分店(委派引擎)
- **店員**:開檯 / 送餐 / 清潔 / **湊咖匹配**(4 細項)
- **廚房**:廚藝 / 出餐 ｜ **保全/顧場**:鎮場處理鬧事
- 公開能力 + **隱藏特質**(面試自我介紹判讀,tells 帶雜訊);士氣→效率;錄用後真相逐步揭露
- **性別**作用在安全軸:👩降鬧事機率但鎮場弱 / 👨鎮場強無預防 → 混編最佳
- 詳見 `GAMEPLAY.md` 決策 3

## E2. 🀄 湊咖系統(麻將館招牌)
一桌要 4 咖;店員用 LINE 群 / **兩雀 APP** 湊桌配咖,填空位=賺檯費,湊不到→客人走。兩雀 APP=可升級的遊戲內匹配工具。

## G. 經營旋鈕

## H. 宣傳系統(增客)✅使用者點名要 → **完整規格 `CODEX_SPEC_promo.md`**
- 核心:投放廣告 → 產生**宣傳熱度(per客層向量)**拉高吸引力,隨時間衰減、需持續投入;每種廣告有各自**主打軸**(客量爆發/網路聲量/常客聲譽/對抗虹吸/低價凹客),逼玩家組合而非刷單招。
- **新增數值 `buzz 網路聲量`**:與「聲譽」分離的網路能見度軸(觀光/散客/學生重、大戶≈0);Google負評/爆滿差評/髒亂被拍→↓,KOL/媒體/社群→↑。進吸引力公式新項 `w8·buzz`。
- **新增客層 `學生`**:客單價最低但衝人氣值,綁大學城區;公車/折價券主帶。
- **型錄 14 筆**(tier 分組,MVP 先做 ★6):
  - 🟢地面:📄傳單★ / 📱社群貼文★ / 🎟️折價券(壓客單價招凹客)
  - 🔵線上:🔍Google關鍵字★(負評→buzz↓) / 📰媒體專訪(店況差反噬) / 🚌公車(學生)
  - 🟡活動:🎉開幕活動 / 🃏店內賽事★(養雀友常客) / 🤝異業聯名(分潤)
  - 🟣高端:📺電視★(**虹吸鄰店客流**/高齡低質) / 🎬KOL★(**爆滿髒亂**) / 🌟名人站台(SML彩蛋) / 🚕計程車隊(回扣/深夜)
  - ⚫灰色:📢網軍刷評(被抓→buzz崩+公關危機+罰)
- 共通防呆:熱度衰減(牙齒sink)/疊放邊際遞減/**容量錯配反噬**(客量>容客量→溢出差評砸buzz)。
- 客素質軸:低質廣告帶進來的客等同「低價客」(髒亂/鬧事/臨檢↑)。

## O. 風水系統 ✅使用者點名要 → **完整規格 `CODEX_SPEC_fengshui.md`**
- 定位:麻將客信風水影響運氣 → 風水達標 = **大戶/雀友磁鐵**(補吸引力缺的「運氣」軸);新增數值 `fengshui` 進公式 `w9`(大戶0.35/雀友0.25高)。
- **完整版擺設謎題**:5 方位槽(東木/南火/中土/西金/北水)× 風水家具(五行/得位/效果)× **五行相生相剋** × **流年吉凶位輪換**(綁賽季→每季要重調=持續管理循環)。
- **風水家具**(型錄14筆,MVP★8):催財/旺丁/化煞/人緣/鎮宅五類 → 連動客單價/來客/降鬧事/buzz/士氣。得位×1.5、失位(被剋)×0.5、相生相鄰+、沖煞歸零、流年財位×1.8/五黃位懲罰(化煞物可鎮)。
- **風水老師**(顧問非常駐,分江湖術士~宗師4級):勘輿掃描/化煞/擇日開市buff/開光/流年調理;江湖術士有騙子風險。
- **風水事件**(併雙向事件):福地口碑buzz+/貴人/財神(利) vs 動土沖煞/員工亂移/五黃入中/假老師破財/客人怪風水砸場(害)。
- 客人運氣感受:高fengshui→客單價/回頭/buzz↑;低fengshui+手氣差→鬧事↑(非賭博下讓「信風水」有實際後果)。
- Phase:家具擺設+計分 Phase 1~2;老師+流年+事件 Phase 3;傳家風水寶物綁 M5 傳承。

## M. 元系統 / 長期進度(本輪定案)

### M1. 老闆等級(Boss Level)
- 靠累積經營表現升級(總營收/存活/分店數綜合)。
- **升級/開新分店 → 解鎖新功能/內容**:新行政區、更高階設備/牌桌、進階服務、管理自動化、更多分店上限相關加成等。

### M2. 連鎖分店(不限間數)
- 可同時經營**多間**分店;**有預算就能開,不限間數**(開新店成本可隨間數遞增以平衡)。
- 需「總部面板」總覽所有分店、各自惰性經營、逐間進去管理。
- 開分店本身是重要成長里程碑(綁 M1 解鎖)。

### M3. 存活天數排行榜
- 全服排行:單館最長存活天數 / 玩家最高總資產 / 最多分店數。
- 倒店會中斷該館存活計時。

### M4. 成就系統
- 徽章里程碑:第一桶金 / 撐過臨檢 / 開第 2·5·10 間分店 / 存活 7·30 天 / 挖角成功 / 削價戰勝利 / 集滿設備…
- 成就可給小幅永久 perk 或稱號徽章。

### M5. 賽季重置 / 輪迴 —「老闆傳承」(✅定案)
- **賽季長度**:**配合 SML 直播賽季**同步(與生態一致,不自訂週期)。
- 每季結束依整季表現(峰值總資產 + 分店數 + 最長存活 + 成就數)換算 **傳承點(聲望)**。
- **傳承強度 = 中等**(老手有感、新人追得上,避免滾雪球輾壓):
  - 歸零:所有館 / 館內金庫 / 個人牙齒經營資產 / 分店 / 排行榜 / 老闆等級 / 資產。
  - **保留**:成就徽章 + 稱號 + 名人堂紀錄(身分榮譽跨季不滅)。
  - 傳承點換**小幅永久加成**:起始資金 ↑ / 開館費折扣 / 全分店客流或收入小幅 % 加成 / **1 件傳家設備 or 1 位老員工**(帶入新季)。加成設**上限**,確保中等強度不變成滾雪球。
- 上季**前三名**進「名人堂」留名 + 專屬稱號/emoji。
- 待細化(Phase 5 實作前):傳承點換算公式、各加成數值與上限、傳家寶挑選 UI。
