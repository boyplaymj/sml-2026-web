- 公開能力 + **隱藏特質**(tells 帶雜訊)/士氣→效率/**性別安全軸**(👩降鬧事鎮場弱/👨鎮場強無預防→混編)/薪資只算排到的班(四班+深夜津貼)。

  - 🎲**游擊中年人**(新):**全天出沒(0-24)**、**打牌最久**(dwell2.0)、**強烈配桌需求**(matchDemand1.5→餵湊咖系統積極填空位)、素質差(quality-0.4)/髒亂/滋事(0.30);**但飽和負外部性**=佔客群比例>門檻(saturationShareThreshold0.25)時,**聲譽/其他客滿意度按超出比例下滑**(少量無妨、變主力毀店);nightmarket/alley 較多。
  - 👴**高齡**:白天(06-15)、**信風水最重**(fengshui0.40)、價格敏感、坐得久 → 退休長輩;高齡社區主客群。
  - 👩‍👧**媽媽/家庭主婦**:活躍 **09-16**(趕回接小孩煮飯,窗短停留短)、**不在意價格**(price0.03)、揪團、重餐飲 → 高客單價;重劃區主客群。
  - 🚸**翹課學生**(新):白天 **09-16** 出沒、**素質差**(quality-0.5)、**可能不付錢(逃單payRisk0.35)/滋事(0.25)** → 白天雙面刃,保全/監視器可防;學校旁主。
- **每客群活躍時段/特性 `balance.clientProfiles`**(9 欄):activeStart/activeEnd(24h,迄<起=跨午夜)/dwell停留倍率/quality素質/troubleRisk滋事/payRisk逃單/**matchDemand配桌需求(餵湊咖)**/**saturationPenalty飽和扣分**/**roundTimeExtend每局配桌+時(hr)→翻桌率↓**。學生 **18-02** 活躍(白天在睡)、**停留最久**(dwell1.6);大戶 20-04 深夜;游擊中年人全天 dwell2.0;麻將新手 roundTimeExtend1。
- **新機制 飽和負外部性**:某客群佔比 > `balance.saturationShareThreshold`(0.25) 時,按超出比例 × 該客群 saturationPenalty 扣聲譽/其他客滿意度(目前只有游擊中年人 0.6);讓「某類客人變主力會毀店」成真。
- **新機制 逃單**:翹課學生消費有機率不付錢→收入損失+滋事;接保全系統(監視器/保全降 payRisk)。

## O. 風水系統 ✅使用者點名要 → **完整規格 `CODEX_SPEC_fengshui.md`**

## M. 元系統 / 長期進度(本輪定案)

### M1. 老闆等級(Boss Level)
- 靠累積經營表現升級(總營收/存活/分店數綜合)。
- **升級/開新分店 → 解鎖新功能/內容**:新行政區、更高階設備/牌桌、進階服務、管理自動化、更多分店上限相關加成等。

### M2. 連鎖分店(不限間數)
- 可同時經營**多間**分店;**有預算就能開,不限間數**(開新店成本可隨間數遞增以平衡)。
- 需「總部面板」總覽所有分店、各自惰性經營、逐間進去管理。
- 開分店本身是重要成長里程碑(綁 M1 解鎖)。

### M3. 存活天數排行榜
- 全服排行:單館最長存活天數 / 玩家最高總資產 / 最多分店數。
- 倒店會中斷該館存活計時。

### M4. 成就系統
- 徽章里程碑:第一桶金 / 撐過臨檢 / 開第 2·5·10 間分店 / 存活 7·30 天 / 挖角成功 / 削價戰勝利 / 集滿設備…
- 成就可給小幅永久 perk 或稱號徽章。

### M5. 賽季重置 / 輪迴 —「老闆傳承」(✅定案)
- **賽季長度**:**配合 SML 直播賽季**同步(與生態一致,不自訂週期)。
- 每季結束依整季表現(峰值總資產 + 分店數 + 最長存活 + 成就數)換算 **傳承點(聲望)**。
- **傳承強度 = 中等**(老手有感、新人追得上,避免滾雪球輾壓):
  - 歸零:所有館 / 館內金庫 / 個人牙齒經營資產 / 分店 / 排行榜 / 老闆等級 / 資產。
  - **保留**:成就徽章 + 稱號 + 名人堂紀錄(身分榮譽跨季不滅)。
  - 傳承點換**小幅永久加成**:起始資金 ↑ / 開館費折扣 / 全分店客流或收入小幅 % 加成 / **1 件傳家設備 or 1 位老員工**(帶入新季)。加成設**上限**,確保中等強度不變成滾雪球。
- 上季**前三名**進「名人堂」留名 + 專屬稱號/emoji。
- 待細化(Phase 5 實作前):傳承點換算公式、各加成數值與上限、傳家寶挑選 UI。
